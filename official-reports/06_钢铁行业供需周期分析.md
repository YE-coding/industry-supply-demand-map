# 钢铁行业供需周期分析

分析日期：2026-07-13 18:21:31 +08:00
地理范围：中国为主
数据时效：2026Q1实际数据 + 2025全年实际数据；2026年5月工业数据滚动校准见文末
行业边界：中国为主、全球矿端参照；覆盖铁矿石、焦煤、焦炭、炼铁炼钢、钢材和地产/基建/制造需求。

## 0. 一句话判断

**中国钢铁行业处于"供强需弱、价格承压"的周期底部阶段，政策调控与绿色转型正在重塑行业格局。**

---

## 1. 产业链

```
上游原料：铁矿石/焦煤/焦炭/废钢
    ↓
炼铁（高炉）
    ↓
炼钢（转炉/电炉）
    ↓
轧钢（热轧/冷轧）
    ↓
钢材产品：螺纹钢/热卷/冷卷/中厚板/型钢
    ↓
下游需求：建筑（房地产/基建）/ 汽车 / 机械 / 家电 / 造船 / 能源
```

**产业链特征**：
- 上游原料高度依赖进口（铁矿石对外依存度约80%）
- 中游冶炼产能严重过剩（粗钢产能约12亿吨，实际需求约9-10亿吨）
- 下游需求结构正在转变（房地产占比下降，制造业占比上升）

## 2. 需求端

### 2.1 需求结构（2025年实际数据）

| 下游领域 | 占比 | 2025年趋势 | 数据来源 |
|---|---|---|---|
| 建筑（房地产+基建） | ~50% | 房地产持续萎缩，基建稳定增长 | 中国钢铁工业协会 |
| 机械 | ~15% | 保持稳定 | 国家统计局 |
| 汽车 | ~8% | 新能源汽车带动增长 | 中汽协 |
| 家电 | ~6% | 稳中有升 | 行业协会 |
| 造船 | ~3% | 订单饱满，景气度高 | 船舶工业协会 |
| 出口 | ~15% | 高位运行，面临贸易摩擦风险 | 海关总署 |

### 2.2 需求端关键数据

**2025年全年数据（实际）**：
- 粗钢表观消费量：约9.2-9.5亿吨（同比下降约2-3%）
- 钢材出口量：约1.1亿吨（同比增长约15%）
- 房地产新开工面积：同比下降约15-20%（持续低迷）
- 汽车产量：约3100万辆（同比增长约5%）

**2026Q1数据（实际）**：
- 粗钢产量：约2.59亿吨（同比增长1.1%）
- 3月单月产量增幅达5%，显示复产意愿较强
- 基建投资保持正增长，但增速放缓

**数据来源**：国家统计局、中国钢铁工业协会、海关总署（2025年年报、2026年4月发布）

### 2.3 需求端趋势判断

- **房地产**：仍处于下行周期，2026年预计继续收缩5-10%，是需求端最大拖累
- **基建**：政策托底作用明显，但地方财政压力制约增速，预计增长3-5%
- **制造业**：汽车、家电、造船等保持稳定增长，成为需求主要支撑
- **出口**：面临欧盟CBAM、美国关税等贸易壁垒，2026年出口增速可能放缓

## 3. 供给端

### 3.1 产能与产量数据

**2025年全年数据（实际）**：
- 粗钢产能：约12-12.5亿吨（产能严重过剩）
- 粗钢产量：约9.8-10亿吨（同比下降约2%）
- 产能利用率：约78-80%（低于合理水平）
- 日均粗钢产量：约270万吨

**2026Q1数据（实际）**：
- 粗钢产量：约2.59亿吨
- 同比增长1.1%
- 3月产量增幅达5%，显示复产压力

**数据来源**：国家统计局（2026年4月发布）、中国钢铁工业协会

### 3.2 供给端关键特征

1. **产能过剩严重**：粗钢产能约12亿吨，实际需求仅9-10亿吨，过剩约20-30%
2. **政策调控持续**：政府自2021年起实施粗钢产量平控/压减政策
3. **产能置换严格执行**：新建产能必须淘汰等量或超量落后产能（部分地区要求1.5:1）
4. **绿色转型加速**：电炉钢比例提升，氢冶金等新技术推进

### 3.3 供给端政策动向

**2025-2026年主要政策**：
- 粗钢产量调控：继续实施"以销定产"，促进供需动态平衡
- 产能置换：严格执行等量/减量置换，淘汰落后产能
- 碳达峰要求：钢铁行业计划2025年前实现碳达峰
- 兼并重组：推动行业集中度提升（目标前10家钢企产量占比达60%）

**数据来源**：工信部、发改委政策文件（2025年发布）、Reuters报道（2025年8月）

## 4. 供需矛盾

### 4.1 核心矛盾

**供强需弱，产能过剩压力持续**

- 供给端：粗钢产能约12亿吨，政策调控下产量仍维持在10亿吨左右
- 需求端：房地产下行拖累，总需求约9-9.5亿吨
- 缺口：供过于求约5000万-1亿吨

| 2026Q1 | 略有改善 | 基建托底，制造业稳定，复产压力显现 |

## 7. 周期阶段

### 7.1 当前阶段判断

**周期底部阶段（衰退末期/复苏初期）**

**判断依据**：
1. **价格**：钢材价格处于历史低位区间，下行空间有限
2. **利润**：行业盈利处于微利或亏损边缘，已触底
3. **库存**：库存水平偏高但边际改善，去化进行中
4. **产能利用率**：约78-80%，低于合理水平但未进一步恶化
5. **政策**：调控政策持续发力，供给端收缩预期增强

### 7.2 周期位置图

```
        繁荣
         ↑
         |    ← 2021年（量价齐升）
         |
         |
         ↓
        衰退
         ↑
         |    ← 2022-2023年（量缩价跌）
         |
         |
         ↓
        底部 ← 2024-2025年（当前位置）
         ↑
         |    ← 2026年？（边际改善）
         |
         ↓
        复苏
```

### 7.3 周期驱动因素

**底部特征**：
- 价格已跌至成本线附近
- 行业亏损面扩大后开始收窄
- 政策调控加码（限产、产能置换）
- 库存开始去化

**复苏信号**：
- 房地产政策放松带动需求回暖
- 基建投资加速
- 钢材价格企稳回升
- 钢厂利润率改善

## 9. 三种情景

### 情景一：乐观情景（概率25%）

**触发条件**：
- 房地产政策大幅放松，销售回暖
- 基建投资超预期加码
- 限产政策严格执行

**结果**：
- 2026年粗钢需求增长2-3%
- 钢材价格上涨10-15%
- 行业盈利显著改善
- 龙头钢企PE修复至15-20倍

### 情景二：基准情景（概率50%）

**触发条件**：
- 房地产继续下行但幅度收窄
- 基建投资保持稳定增长
- 限产政策适度执行

**结果**：
- 2026年粗钢需求持平或微降
- 钢材价格小幅波动（±5%）
- 行业盈利维持微利状态
- 钢铁板块估值小幅修复

### 情景三：悲观情景（概率25%）

**触发条件**：
- 房地产继续大幅下行
- 贸易摩擦加剧，出口受阻
- 限产政策执行不力

**结果**：
- 2026年粗钢需求下降3-5%
- 钢材价格下跌10-15%
- 行业亏损面扩大
- 部分中小钢企退出市场

## 11. 最终说明

### 11.1 分析局限性

1. **数据时效性**：部分数据为2025年全年或2026Q1数据，可能无法完全反映最新变化
2. **预测不确定性**：2026年预测数据基于当前趋势，实际可能受政策、外部环境等影响
3. **数据来源**：部分数据来自二手来源，可能存在偏差
4. **搜索限制**：由于搜索工具限制，部分最新数据可能未获取到

### 11.2 核心结论

1. **周期位置**：钢铁行业处于周期底部阶段，下行空间有限，但复苏仍需时间
2. **供需格局**：供强需弱格局短期难以改变，政策调控是关键变量
3. **价格判断**：钢材价格已处于低位，2026年预计小幅波动，难有大幅上涨
4. **验证框架**：以铁水—表需—库存—吨钢利润的闭环确认真实复苏；行业供需结论不构成任何证券或交易建议。

### 13.5 Supply-Demand Conflict and Transmission

核心矛盾不是“需求强”本身，而是地产需求偏弱与制造业需求增长之间的结构分化能否在需求兑现前形成有效供给。

```text
需求/预算变化 -> 建筑/制造需求订单 -> 地产需求偏弱与制造业需求增长之间的结构分化交期与价格 -> 收入/毛利 -> 扩产 -> 有效产能释放 -> 紧张缓解或过剩
```

- 立即受益：已有有效产能、通过认证且订单可转收入的节点。
- 滞后受益：等待扩产、良率爬坡、并网或客户认证的节点。
- 可能非受益：只有规划产能、概念关联或无法转嫁成本的节点。
