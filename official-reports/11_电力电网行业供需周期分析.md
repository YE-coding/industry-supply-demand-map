## 0. 一句话判断

**电力行业处于"结构性扩张期"：需求稳步增长（2025年用电量首破10万亿kWh），供给端新能源装机历史性超越煤电，火电盈利因煤价下行显著修复，电网投资进入"十五五"超级周期。整体供需宽松，但季节性/区域性紧张仍存。**

- 用电侧：AI数据中心、新能源汽车成为新增量，第三产业和居民用电增速领先

## 2. 需求端

### 2.1 2025全年用电量

- **全社会用电量：首次突破10万亿kWh**，中国成为全球单一国家最大电力消费国
- 同比增速：约5.5-6%
- 人均用电量：约7,100 kWh

**用电结构（2025年）：**
| 产业 | 占比 | 增速特征 |
|---|---|---|
| 第二产业（工业） | ~65% | 稳定增长，高耗能产业向绿电转型 |
| 第三产业（商业/服务） | ~18% | 增速较快，数据中心贡献显著 |
| 居民生活 | ~15% | 受气候和电气化驱动 |
| 第一产业（农业） | ~2% | 平稳 |

### 2.2 2026Q1用电量

- **Q1 2026全社会用电量：2.51万亿kWh，同比增长5.2%**
- 1-2月增速6.1%，3月略有放缓
- 4月用电量约8,205亿kWh（CEIC数据）
- 总能源消费Q1增长3.9%，较2025全年加速0.4个百分点

### 2.3 需求驱动因素

1. **AI/数据中心**：高耗能、24小时不间断，成为电力需求新增长极
2. **新能源汽车**：电动化渗透率持续提升，充电桩用电快速增长
3. **工业升级**：高端制造业用电需求结构性增长
4. **极端气候**：2025年7月用电量创历史新高（10,226亿kWh）

## 3. 供给端

### 3.1 装机容量（2025年底）

| 电源类型 | 装机容量 | 占比 | 同比变化 |
|---|---|---|---|
| **风电+光伏** | **18.4亿kW (1,840 GW)** | **~48%** | **历史性超越煤电** |
| 煤电（火电） | 15.39亿kW (1,539 GW) | ~40% | 增速放缓 |
| 水电 | ~4.2亿kW | ~11% | 平稳 |
| 核电 | 1.2亿kW (120 GW) | ~3% | 全球第一 |
| **合计** | **~38亿kW** | 100% | |

**关键里程碑：**
- 2025年新增风电+光伏装机约375 GW，其中光伏315 GW（创纪录）
- 风电+光伏装机首次超过煤电，差距约300 GW
- 非化石能源装机占比超过55%

### 3.2 发电量结构（2025年）

| 电源类型 | 发电量占比 | 变化趋势 |
|---|---|---|
| 煤电 | ~58% | **首次绝对下降**（减少71 TWh，2015年以来首次） |
| 风电+光伏 | ~22% | 快速增长 |
| 水电 | ~13-14% | 受来水影响波动 |
| 核电 | ~5% | 稳步增长 |
| 其他 | ~2% | |

**历史性变化：2025年化石能源发电占比降至58%（2024年约62%），煤电发电量绝对值首次下降。**

### 3.3 核电发展

- 2025年底核电装机超1.2亿kW，全球第一
- 58座运行中反应堆，装机59.9 GW
- 29座在建中
- 2025年新增1.1 GW，2026年（截至5月）新增2.2 GW
- 2025年批准10座新反应堆
- 中国核电容量自2016年以来增长76%（+24 GW）

### 3.4 2026年供给规划

- 国家能源工作会议：2026年新增风电+光伏超2亿kW（200 GW）
- 风电+光伏装机预计2026年底占总装机约50%
- 新型储能产业加速发展

## 4. 供需矛盾

### 4.1 整体格局：**供给充裕，结构性矛盾突出**

| 矛盾类型 | 表现 | 严重程度 |
|---|---|---|
| **总量平衡** | 装机容量远超峰值负荷 | 低风险 |
| **季节性紧张** | 夏季迎峰度夏仍有缺口风险 | 中等 |
| **区域不平衡** | 西部新能源富余 vs 东中部负荷中心 | 持续 |
| **时段性波动** | 新能源出力不稳定，晚高峰缺电 | 中等 |
| **煤电角色转型** | 利用小时数持续下降，但需保供 | 结构性 |

### 4.2 核心矛盾

**新能源高速增长 vs 电网消纳能力不足**
- 新能源装机爆发式增长，但弃风弃光率在部分地区仍较高
- 储能配套不足，调峰能力有限
- 特高压通道建设滞后于新能源装机速度

## 7. 周期阶段

### 判断：**结构性扩张期（中后段）**

**依据：**

1. **需求侧**：用电量保持5-6%稳健增长，尚未见顶。AI/数据中心/新能源汽车提供新增量。
2. **供给侧**：新能源装机爆发式增长，2025年是历史性拐点（新能源装机超煤电）。但增速可能在2026-2027年边际放缓。
3. **价格信号**：煤价下行、火电盈利修复属于"周期复苏"特征；新能源电价承压属于"产能扩张"特征。
4. **资本开支**：电网投资进入"十五五"超级周期，资本开支大幅增长。
5. **政策驱动**："双碳"目标、全国统一电力市场建设、新型电力系统构建，政策确定性高。

**周期位置：**
- 火电：盈利修复期（煤价下行受益），但长期面临转型压力
- 新能源发电：高速扩张期（装机高增），但利润率承压
- 核电：稳定增长期（审批确定性高）
- 水电：成熟期（来水周期影响大）
- 电网：超级投资周期（"十五五"4万亿规划）

## 9. 三种情景

### 情景一：基准情景（概率55%）

**"新能源持续高增，电网投资加速落地"**
- 2026年用电量增速5-5.5%
- 新能源新增装机200+ GW
- 煤价中枢维持700-750元/吨
- 国网投资按计划推进
- **投资含义**：标配电力板块，超配电网设备+核电

### 情景二：乐观情景（概率25%）

**"AI算力需求爆发+极端高温"**
- 2026年用电量增速超6%
- 夏季电力紧张推高电价
- 煤价季节性反弹
- **投资含义**：火电弹性标的受益，新能源运营商电价改善

### 情景三：悲观情景（概率20%）

**"经济放缓+新能源产能过剩"**
- 2026年用电量增速降至4%以下
- 新能源弃电率上升
- 煤价进一步下行但火电利用小时数也下降
- **投资含义**：防御性配置核电+水电，规避新能源制造端

## 11. 最终说明

### 核心结论

中国电力行业正处于**结构性转型的关键拐点期**：

1. **历史性突破**：2025年风电+光伏装机首次超过煤电，用电量首破10万亿kWh，煤电发电量绝对值首次下降。这三个"首次"标志着中国电力系统正在发生根本性变化。

2. **盈利分化**：火电企业因煤价下行实现盈利修复（五大发电净利润创历史新高384亿元），但新能源运营商面临电价下行压力。行业内部盈利分化加剧。

3. **电网超级周期**：国网"十五五"规划投资4万亿元（较"十四五"+40%），特高压设备进入加速招标期，电网设备产业链迎来确定性增长。

4. **核电确定性最高**：年均6-8座新堆审批节奏稳定，利用小时数高达7,000+小时，是电力板块中确定性最高的子行业。

### 主要风险

- 新能源装机增速过快导致弃电率上升
- 极端天气对电力供需平衡的冲击
- 电力市场化改革进度不及预期
- 煤价超预期反弹压缩火电利润

### 13.5 Supply-Demand Conflict and Transmission

核心矛盾不是“需求强”本身，而是跨区输电、配电和并网工程的实际建设进度能否在需求兑现前形成有效供给。

```text
需求/预算变化 -> 用电负荷订单 -> 跨区输电、配电和并网工程的实际建设进度交期与价格 -> 收入/毛利 -> 扩产 -> 有效产能释放 -> 紧张缓解或过剩
```

- 立即受益：已有有效产能、通过认证且订单可转收入的节点。
- 滞后受益：等待扩产、良率爬坡、并网或客户认证的节点。
- 可能非受益：只有规划产能、概念关联或无法转嫁成本的节点。
