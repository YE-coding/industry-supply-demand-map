# 机器人行业供需周期分析

分析日期：2026-07-13 18:21:31 +08:00
地理范围：全球（重点关注中国）
数据时效：2025全年实际数据 + 2026年5月工业数据；滚动校准见文末
行业边界：全球与中国；覆盖减速器、伺服、控制器、传感器、芯片、本体、系统集成和工业/服务终端，区分工业机器人与人形机器人。

---

## 0. 一句话判断

**机器人行业处于"工业机器人成熟增长+人形机器人导入期"的双轨并行阶段。2025年全球工业机器人安装量54.2万台（连续4年超50万台），中国人形机器人出货约1.3万台（同比+400%），但人形机器人仍处于"有货无场景"的早期验证阶段，量产规模小、成本高、应用场景有限。工业机器人是确定性较高的增长赛道，人形机器人是高赔率但高不确定性的长期主题。**

## 1. 产业链

### 1.1 工业机器人产业链

```text
核心零部件 --> 本体制造 --> 系统集成 --> 终端应用
    |              |            |            |
 减速器/伺服    ABB/发那科   ABB/发那科    汽车制造
 电机/控制器   安川/库卡    安川/库卡     3C电子
 传感器        埃斯顿/汇川   埃斯顿/拓斯达  锂电/光伏
              新时达/埃夫特               食品/医药
```

### 1.2 人形机器人产业链

```text
AI大模型 --> 计算平台 --> 执行器/关节 --> 传感器 --> 本体制造 --> 应用场景
    |           |            |            |          |           |
 OpenAI      NVIDIA       电机/减速器   力传感器    AgiBot      工厂巡检
 Google      高通          丝杠/轴承     视觉传感器   Unitree     仓储物流
 华为        地平线        灵巧手       IMU/编码器   Tesla       家庭服务
                           电池                      Figure      危险环境
```

**关键瓶颈（按严重程度排序）：**
1. **AI泛化能力**：人形机器人需要在非结构化环境中自主决策，当前AI模型能力不足
2. **成本控制**：人形机器人单台成本$6K-$14K（中国），远高于工业机器人
3. **可靠性/耐久性**：人形机器人需要长时间稳定运行，当前可靠性不足
4. **场景验证**：缺乏规模化商业应用场景验证
5. **核心零部件**：精密减速器、力矩传感器、灵巧手等依赖进口

## 4. 供需矛盾

### 4.1 工业机器人：供需基本平衡

**需求侧：** 全球安装量54.2万台，连续4年超50万台，需求稳定增长

**供给侧：** 发那科/ABB/安川等巨头产能充足，中国厂商快速扩产

**矛盾：** 核心零部件（精密减速器、高端伺服）仍有供给瓶颈，但整体供需基本平衡

### 4.2 人形机器人：有货无场景

**需求侧：** 市场热度高，但实际商业需求有限
- 工厂巡检、仓储物流等场景仍处于验证阶段
- 缺乏规模化采购的动力

**供给侧：** 中国厂商快速出货，但：
- 出货量小（全球仅1.3万台）
- 成本高（$6K-$14K）
- 可靠性/耐久性不足
- AI泛化能力有限

**矛盾本质：** 人形机器人的"供给"（出货量）超前于"需求"（商业场景验证），存在"有货无场景"的风险

### 4.3 矛盾传导链

```
AI技术突破 --> 人形机器人概念爆发 --> 资本涌入 --> 厂商快速出货
         |                                          |
    但商业场景未验证 --> 实际需求有限 --> 库存积压风险
         |                                          |
    核心零部件不成熟 --> 成本高/可靠性差 --> 客户观望
         |                                          |
    价格战开始（$6K起） --> 利润率压缩 --> 弱势厂商退出
```

## 7. 周期阶段

### 7.1 判断：工业机器人=成熟增长期；人形机器人=导入期

**工业机器人：**

| 维度 | 当前状态 | 典型特征 |
|------|---------|---------|
| 安装量 | 54.2万台/年 | 稳定增长 |
| 增速 | +6% YoY | 成熟行业 |
| 竞争格局 | 寡头垄断 | 发那科/ABB/安川主导 |
| 中国国产替代 | 份额57% | 加速中 |
| 周期阶段 | **成熟增长期** | - |

**人形机器人：**

| 维度 | 当前状态 | 典型特征 |
|------|---------|---------|
| 出货量 | 1.3万台/年 | 极小规模 |
| 增速 | +400% YoY | 高速增长但基数小 |
| 商业化 | 早期验证 | 有货无场景 |
| 成本 | $6K-$14K | 远高于工业机器人 |
| AI能力 | 有限 | 泛化能力不足 |
| 周期阶段 | **导入期** | - |

### 7.2 与历史周期的类比

**人形机器人类似于：**
- **2010年智能手机早期**：iPhone 4发布后，安卓手机厂商涌入，但生态尚未成熟
- **2015年新能源汽车早期**：政策驱动+资本涌入，但产品力不足

**关键差异：**
- 人形机器人的AI技术依赖度更高，技术突破节奏不确定
- 工业机器人有明确的ROI，人形机器人的ROI尚未验证
- 中国厂商在价格上具有绝对优势

## 9. 三种情景

### 情景A：人形机器人商业化加速（概率20%）

**假设：** AI泛化能力突破，人形机器人在工厂/仓储场景验证成功

**推演：**
- 2027年全球人形机器人出货量突破10万台
- Tesla Optimus量产，带动行业热度
- 核心零部件需求爆发，供应链受益
- 人形机器人板块估值大幅扩张

**投资含义：** 看多AgiBot/Unitree/Tesla、核心零部件（减速器/传感器/灵巧手）

### 情景B：工业机器人稳定+人形缓慢渗透（概率55%）

**假设：** 工业机器人维持5-8%增长，人形机器人小规模验证但未大规模商用

**推演：**
- 工业机器人安装量维持55-60万台/年
- 人形机器人出货量缓慢增长至5-10万台
- 核心零部件需求稳步增长
- 行业估值合理

**投资含义：** 选择工业机器人龙头+核心零部件国产替代标的

### 情景C：人形机器人泡沫破裂（概率25%）

**假设：** AI能力不足，人形机器人商业化进度远低于预期

**推演：**
- 人形机器人出货量停滞在1-2万台
- 资本退出，估值大幅回调
- 弱势厂商退出，行业整合
- 工业机器人仍稳定增长

**投资含义：** 规避纯人形机器人概念股，关注工业机器人基本面

### 13.5 Supply-Demand Conflict and Transmission

核心矛盾不是“需求强”本身，而是AI泛化能力：人形机器人需要在非结构化环境中自主决策，当前AI模型能力不足能否在需求兑现前形成有效供给。

```text
需求/预算变化 -> 数据/软件服务订单 -> AI泛化能力：人形机器人需要在非结构化环境中自主决策，当前AI模型能力不足交期与价格 -> 收入/毛利 -> 扩产 -> 有效产能释放 -> 紧张缓解或过剩
```

- 立即受益：已有有效产能、通过认证且订单可转收入的节点。
- 滞后受益：等待扩产、良率爬坡、并网或客户认证的节点。
- 可能非受益：只有规划产能、概念关联或无法转嫁成本的节点。
