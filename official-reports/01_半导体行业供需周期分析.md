# 半导体行业供需周期分析

分析日期：2026-07-13 18:21:31 +08:00
地理范围：全球
数据时效：2025全年实际数据 + 2026Q1实际数据；2026-07-12滚动校准见文末
行业边界：全球与中国；覆盖材料、设备、设计、晶圆制造、封装测试和终端需求，成熟制程与先进制程分开判断。

---

## 0. 一句话判断

**半导体行业处于"AI驱动的结构性扩张中期"，2025年全球销售额达$791.7B（+25.6% YoY），2026年预计突破$1万亿大关。AI算力需求（GPU/HBM/先进封装）是核心增长引擎，传统消费电子需求温和复苏，但存储器产能向AI倾斜导致常规DRAM/NAND出现结构性短缺。成熟制程产能过剩与先进制程供不应求并存，行业呈现"冰火两重天"格局。**

## 1. 产业链

```text
硅材料/特种气体 --> 设备(光刻/刻蚀/沉积) --> 晶圆制造 --> 先进封装 --> 测试 --> 芯片设计 --> 终端应用
    |                   |                    |            |           |         |            |
  信越/SUMCO        ASML/应用材料        台积电/三星      台积电CoWoS   长电科技   NVIDIA/AMD    AI/汽车/IoT
  关东电工          东京电子/泛林        英特尔/中芯      ASE/日月光    通富微电   高通/联发科    消费电子/工业
  杜邦              KLA                 华虹/GlobalFoundries                      苹果/华为
```

**关键瓶颈（按严重程度排序）：**
1. **先进制程产能**：3nm/5nm制程被台积电垄断（纯代工72%份额），三星/英特尔良率不足
2. **EUV光刻机**：ASML独家供应，交期18-24个月，High-NA EUV更稀缺
3. **HBM存储**：AI GPU核心配套，供给增速62%但需求增速更高
4. **先进封装（CoWoS）**：产能从13K→125K wpm扩张10倍仍供不应求
5. **特种气体/材料**：氖气、光刻胶等关键材料供应链脆弱

## 4. 供需矛盾

### 4.1 核心矛盾：先进制程供不应求 + 成熟制程结构性过剩

**先进制程（<7nm）：**
- 台积电3nm/5nm产能被NVIDIA/苹果/AMD锁定
- CoWoS封装产能扩张10倍（13K→125K wpm）仍不足
- EUV光刻机交期18-24个月，限制产能扩张速度

**成熟制程（≥28nm）：**
- 中国大量扩产（中芯、华虹、晶合集成等）
- 全球成熟制程产能过剩风险上升
- 车规/工控芯片需求稳定但增长有限

### 4.2 矛盾传导链

```
AI算力需求爆发 --> GPU/ASIC芯片需求激增 --> 先进制程产能紧张
         |                                       |
    HBM需求暴增 --> 存储器产能向AI倾斜 --> 常规DRAM/NAND短缺
         |                                       |
    CoWoS封装需求 --> 封装产能瓶颈 --> 芯片交付延迟
         |                                       |
    EUV光刻机需求 --> ASML产能受限 --> 扩产速度受限
```

### 4.3 矛盾演变方向

- **短期（2026H2）：** AI芯片供需缺口仍然存在，但边际改善。HBM供给增速首次超过需求增速。
- **中期（2027-2028）：** 2nm制程量产（台积电N2、三星SF2），先进制程产能显著增加。
- **风险：** 中国成熟制程产能大规模释放可能导致全球产能过剩，价格战加剧。

## 7. 周期阶段

### 7.1 判断：AI驱动的结构性扩张中期

**判断依据：**

| 维度 | 当前状态 | 典型特征 |
|------|---------|---------|
| 市场规模增速 | +25.6%（2025），+23-53%（2026E） | 历史性高增长 |
| AI需求增速 | +85-92% YoY | 远超行业平均 |
| 供给约束 | 先进制程/CoWoS/EUV | 结构性瓶颈 |
| 价格信号 | HBM降30%，常规DRAM涨 | 结构性分化 |
| 竞争格局 | 台积电垄断先进代工 | 高度集中 |
| 产能扩张 | 全球大规模扩产 | 扩张中期 |

### 7.2 与历史周期的类比

当前周期类似于：
- **2017-2018年内存超级周期**：AI需求驱动HBM类似当年云服务驱动DRAM
- **1995-2000年互联网基础设施周期**：资本开支超前于收入，但需求增长验证了投资

**关键差异：**
- 当前AI算力需求的"买单方"是盈利能力极强的超大规模厂商
- 中国国产替代创造了额外的结构性需求
- 先进制程技术壁垒极高，产能扩张速度受限

## 9. 三种情景

### 情景A：AI需求持续超预期（概率30%）

**假设：** Agentic AI推理需求爆发，AI应用商业化加速，超大规模厂商Capex再上调

**推演：**
- 2026年全球半导体市场可能突破$1.3T（IDC预测）
- 先进制程产能更加紧张，台积电毛利率维持高位
- HBM短缺延长至2027-2028年
- 存储器板块整体受益

**投资含义：** 看多NVIDIA、台积电、SK Hynix、ASML

### 情景B：AI需求稳健+传统需求复苏（概率50%）

**假设：** AI增速从90%+回落至50-60%，汽车/工业/消费电子温和复苏

**推演：**
- 2026年全球半导体市场~$975B（WSTS预测）
- 先进制程供需逐步平衡
- 成熟制程价格战加剧但可控
- 行业整体利润增速放缓但仍正增长

**投资含义：** 选股分化，关注估值合理的二线受益者

### 情景C：AI商业化回报低于预期（概率20%）

**假设：** AI应用商业化进度慢于资本开支增速，超大规模厂商Capex回报周期拉长

**推演：**
- 2027年可能出现先进制程产能过剩
- NVIDIA增速放缓至20-30%，估值承压
- 存储器价格大幅下跌
- 半导体设备订单减少

**投资含义：** 科技股估值回调，行业进入下行周期

### 11.2 核心结论

1. **AI是半导体行业的核心增量引擎：** 2025年$791.7B→2026年$975B-$1.29T的增长中，AI贡献了50%以上的增量。

4. **中国国产替代加速：** 自给率从16%→28%，$150B补贴驱动，但先进制程仍受制于EUV设备。

5. **最大风险是产能过剩：** 中国成熟制程扩产+全球先进制程扩张，2027-2028年可能面临周期性调整。

### 13.5 Supply-Demand Conflict and Transmission

核心矛盾不是“需求强”本身，而是先进制程产能：3nm/5nm制程被台积电垄断（纯代工72%份额），三星/英特尔良率不足能否在需求兑现前形成有效供给。

```text
需求/预算变化 -> 应用订单 -> 先进制程产能：3nm/5nm制程被台积电垄断（纯代工72%份额），三星/英特尔良率不足交期与价格 -> 收入/毛利 -> 扩产 -> 有效产能释放 -> 紧张缓解或过剩
```

- 立即受益：已有有效产能、通过认证且订单可转收入的节点。
- 滞后受益：等待扩产、良率爬坡、并网或客户认证的节点。
- 可能非受益：只有规划产能、概念关联或无法转嫁成本的节点。
