# 焦煤行业供需周期分析

分析日期：2026-07-13 18:21:31 +08:00
地理范围：中国为主
数据时效：2026Q1实际数据 + 2025全年实际数据；2026年5月进口数据滚动校准见文末
行业边界：中国为主、海运焦煤为补充；覆盖开采、洗选、焦化、焦炭和高炉需求，不与动力煤混同。

## 0. 一句话判断
焦煤行业正处于**供需再平衡阶段**，供给端受政策调控约束，需求端受钢铁行业减产压制，2026年预计呈现"供给小幅收紧、需求同步下降"的弱平衡格局，价格将维持区间震荡，缺乏趋势性行情。

## 1. 产业链
```text
焦煤开采 -> 洗选 -> 焦炭 -> 炼铁 -> 炼钢 -> 下游（建筑/汽车/机械）
```
- **上游**：焦煤开采（山西、内蒙古、安徽等主产区）
- **中游**：洗选加工 -> 焦炭生产
- **下游**：钢铁冶炼 -> 建筑/汽车/机械/家电等终端需求

## 2. 需求端

### 2.1 钢铁行业需求
- **2025年实际数据**：中国粗钢产量维持高位，但增速放缓
- **2026年预期**：受钢铁行业减产政策影响，铁水产量预计减少700-800万吨，焦煤需求将同步下降
- **终端需求**：建筑、汽车、机械等下游行业需求疲软，钢材出口面临贸易摩擦压力

### 2.2 焦炭生产需求
- 焦炭产能过剩格局未改，焦化企业利润微薄
- 春节前后下游补库需求提供短期支撑，但持续性不足
- 焦炭价格受制于焦煤成本与钢铁需求双重挤压

### 2.3 需求结构变化
- **占比**：焦煤消费中，钢铁行业占比约85%，化工及其他占15%
- **趋势**：电炉钢比例提升将长期压制焦煤需求，但短期影响有限

**数据来源**：华经产业研究院、南华期货年度展望（2026年1月发布）

## 3. 供给端

### 3.1 国内产量
- **2025年实际数据**：
  - 炼焦原煤产量：约130,785万吨（实际数据，国家统计局2026年1月发布）
  - 炼焦精煤产量：约47,952万吨（实际数据）
- **2026年预测**：
  - 预计全国焦煤产量同比增长1.3%（预测数据，雪球研究2026年1月）
  - 另有机构预测国内焦煤产量将同比小幅收紧-0.5%（预测数据，南华期货2025年12月）
  - 核心增量来自山西、内蒙古等主产区的产能优化

### 3.2 进口情况
- **2023年实际数据**：累计进口炼焦煤1.02亿吨，同比增长61.3%，创历史最高纪录
- **2024年实际数据**：高位回落
- **2025年实际数据**：同比下降3.0%
- **2026年Q1实际数据**：出现回升迹象
- **2026年预测**：预计全国焦煤进口量将达1.19亿吨，同比增加300-400万吨（预测数据，东方财富2025年12月）
- **进口依赖度**：年均进口炼焦煤超1亿吨，占总供应的比重高达20%

### 3.3 产能与政策约束
- **安全检查**：2025年7月国家能源局对8省区开展煤矿超产核查，导致8-9月炼焦煤单月产量同比下降10.9%、9.5%
- **环保限产**："查超产"、"控能耗"等行政手段约束过量供应
- **产能储备**：建立产能储备机制平抑供应波动
- **山西焦煤集团**：2026年1月召开安全环保工作会议，强调安全生产

**数据来源**：华经产业研究院（2026年3月）、雪球研究（2026年1月）、南华期货（2025年12月）

## 4. 供需矛盾

### 4.1 供需平衡分析
- **2026年预测**：焦煤市场预计整体趋于供需平衡（预测数据，东证期货2025年12月）
- **供给增量**：进口增量与国产增量叠加，全年供应总量将增加约800万吨
- **需求减量**：受钢铁行业减产影响，焦煤需求将同步下降
- **库存压力**：供需缺口逆转为过剩，库存累积压力显现

### 4.2 结构性矛盾
- **优质焦煤稀缺**：优质主焦煤供需缺口边际走阔，价值重估可期
- **进口依赖**：进口煤对国内市场价格形成压制，澳煤作为海运市场的关键定价基准
- **政策调控**：反内卷政策可能通过调整国内焦煤产量，动态平衡焦煤供需

### 4.3 区域差异
- **山西**：主产区，受安全检查影响较大
- **内蒙古**：产能释放较快，增量贡献显著
- **进口来源**：蒙古、俄罗斯、澳大利亚为主要进口来源国

**数据来源**：东证期货（2025年12月）、雪球研究（2026年1月）

## 6. 受益者与非受益者

### 6.1 受益者
- **优质焦煤生产企业**：拥有优质主焦煤资源的企业，如山西焦煤、平煤股份等
- **进口贸易商**：进口煤贸易量增加，套利窗口存在
- **大型煤炭集团**：政策支持产能整合，集中度提升

### 6.2 非受益者
- **中小焦煤企业**：受安全检查、环保限产影响较大，产能释放受限
- **焦化企业**：成本端受焦煤价格支撑，需求端受钢铁行业压制，利润空间被挤压
- **钢铁企业**：虽然焦煤价格稳定，但下游需求疲软，整体盈利承压

## 7. 周期阶段

**当前阶段：供需再平衡期（弱平衡）**

- **供给端**：政策调控下的产能约束与进口补充并存，整体供给趋稳
- **需求端**：钢铁行业减产压制需求，终端需求弹性缺失
- **价格特征**：区间震荡，缺乏趋势性行情
- **库存周期**：从去库存向补库存过渡，但补库力度有限

**周期位置**：处于下行周期末段的磨底阶段，等待需求端企稳信号

## 9. 三种情景

### 9.1 基准情景（概率60%）
- **假设**：钢铁行业减产幅度有限，焦煤进口稳定增长
- **结果**：焦煤价格维持1400-1600元/吨区间震荡，全年均价较2025年小幅下降3-5%
- **驱动**：供需弱平衡，政策调控为主

### 9.2 乐观情景（概率20%）
- **假设**：稳增长政策发力，钢铁需求超预期回升
- **结果**：焦煤价格突破1600元/吨，优质焦煤价格涨幅更大
- **驱动**：需求端改善，库存去化加速

### 9.3 悲观情景（概率20%）
- **假设**：全球经济衰退，钢铁需求大幅下滑
- **结果**：焦煤价格跌破1400元/吨，行业进入深度亏损
- **驱动**：需求端崩塌，库存被动累积

### 13.5 Supply-Demand Conflict and Transmission

核心矛盾不是“需求强”本身，而是钢厂和焦化厂真实开工需求能否在需求兑现前形成有效供给。

```text
需求/预算变化 -> 炼钢订单 -> 钢厂和焦化厂真实开工需求交期与价格 -> 收入/毛利 -> 扩产 -> 有效产能释放 -> 紧张缓解或过剩
```

- 立即受益：已有有效产能、通过认证且订单可转收入的节点。
- 滞后受益：等待扩产、良率爬坡、并网或客户认证的节点。
- 可能非受益：只有规划产能、概念关联或无法转嫁成本的节点。
