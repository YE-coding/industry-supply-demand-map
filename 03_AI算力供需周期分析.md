# AI算力供需周期分析

分析日期：2026-07-13 18:21:31 +08:00
地理范围：全球
数据时效：2026Q1实际数据 + 2025全年实际数据；2026-07-12滚动校准见文末
行业边界：全球与中国；覆盖云厂商资本开支、GPU/ASIC、HBM、先进封装、服务器、网络、电力和散热，不等同于全部人工智能应用。

---

## 0. 一句话判断

**AI算力行业处于超级周期的"加速扩张中期"，需求端以85-92%的同比增速持续爆发，供给端受GPU产能（台积电CoWoS封装）、HBM存储和电力三大瓶颈制约，供需缺口在2026年仍难以根本弥合，但供给增速首次超过需求增速的拐点信号已经出现。**

---

## 1. 产业链

```text
GPU/ASIC --> HBM --> 先进封装(CoWoS) --> AI服务器 --> 交换机/网络 --> 电力 --> 散热 --> 数据中心 --> 模型/应用需求
   |           |           |                |              |           |        |          |
  NVIDIA     SK Hynix     TSMC           Dell/Foxconn   Broadcom    电力公司   液冷方案    CSP/企业
  AMD        Micron       CoWoS-L/S      Super Micro    Cisco       核能/风光  散热厂商    超大规模
  Google TPU  Samsung                     HPE                         电网      数据中心    模型厂商
  AWS Trainium
```

**关键瓶颈（按严重程度排序）：**
1. **GPU产能**：台积电先进制程（3nm/5nm）+ CoWoS封装产能是核心约束
2. **HBM存储**：SK Hynix/Micron 2026年产能已全部售罄，短缺预计持续到2027年
3. **先进封装**：CoWoS是唯一经过量产验证的2.5D封装技术，NVIDIA锁定了大部分产能至2027年
4. **电力供应**：Microsoft有$800亿未完成Azure订单积压，主要受电力限制；IEA预测2022-2026年数据中心电力消耗翻倍

## 4. 供需矛盾

### 4.1 核心矛盾：需求增速远超供给弹性

**需求侧：** NVIDIA数据中心收入单季$75.2B，同比+92%；全球服务器市场收入$1,226亿/季，同比+30.4%

**供给侧：**
- CoWoS产能从72.5K wpm（End 2025）目标扩至125K wpm（End 2026），增幅约72%
- HBM供给位元增长+62% YoY
- 但需求增长（按NVIDIA收入计）高达85-92%

**矛盾本质：** 需求增速（85-92%）> 供给增速（62-72%），缺口仍在扩大。

### 4.2 三大瓶颈的传导链

```
台积电CoWoS产能不足 --> GPU封装受限 --> GPU出货量受限 --> AI服务器交付延迟
         |                                                      |
HBM供给紧张（全部售罄） --> GPU显存受限 --> 高端GPU供给受限         |
         |                                                      |
电力供应不足 --> 数据中心建设延迟 --> Microsoft $800亿订单积压 <---+
```

### 4.3 矛盾的演变方向

- **短期（2026H2）：** 供需缺口仍然存在，但边际改善。HBM供给增速首次超过需求增速是重要拐点信号。
- **中期（2027）：** 随着Samsung HBM4量产、CoWoS产能达125K+ wpm、Micron新加坡工厂投产，供给瓶颈有望显著缓解。
- **风险：** 如果AI推理需求超预期爆发（Agentic AI），需求侧可能再次加速，延长供需紧张周期。

## 7. 周期阶段

### 7.1 判断：加速扩张中期（Mid-Acceleration）

**判断依据：**

| 维度 | 当前状态 | 典型特征 |
|------|---------|---------|
| 需求增速 | +85-92% YoY | 远超GDP增速，持续加速 |
| 供给增速 | +62-72% YoY | 快速扩张但仍有缺口 |
| 资本开支 | $685B-$715B（五大厂商） | 历史性规模，几乎翻倍 |
| 价格 | HBM下降30%，GPU毛利率稳定 | 供给改善但定价权仍在 |
| 库存 | NVIDIA库存+20.6% QoQ | 正常备货，非堆积 |
| 竞争格局 | ASIC首次超过GPU增速 | 多元化萌芽，但NVIDIA仍主导 |
| 瓶颈 | CoWoS/HBM/电力 | 供给约束仍紧但边际改善 |

### 7.2 与历史周期的类比

当前AI算力周期类似于：
- **2010-2012年智能手机周期的"加速扩张中期"：** 需求爆发式增长，供应链快速扩张但仍有缺口，核心供应商（如台积电、三星）利润丰厚
- **1996-1998年互联网基础设施周期：** 资本开支大幅超前于收入，但需求增长验证了投资的合理性

**关键差异：**
- 当前AI算力需求的"买单方"是盈利能力极强的超大规模厂商（非亏损初创），资本开支可持续性更高
- AI推理需求正在从训练需求中独立出来，形成第二增长曲线

## 9. 三种情景

### 情景A：需求持续超预期（概率30%）

**假设：** Agentic AI推理需求爆发，AI应用商业化加速，云服务收入增速提升至30%+

**推演：**
- 2026H2-2027年数据中心Capex可能再上调20-30%
- 供需缺口扩大，CoWoS/HBM瓶颈更加紧张
- NVIDIA可能上调全年指引，毛利率维持75%+
- 电力瓶颈成为更严重的约束

**投资含义：** 看多NVIDIA、TSMC、SK Hynix；电力/核能概念持续走强

### 情景B：供需逐步平衡（概率50%）

**假设：** 需求增速从90%+回落至50-60%，供给增速维持60-70%，2027年供需基本平衡

**推演：**
- 2026H2 HBM价格继续下降，但幅度收窄
- NVIDIA增速放缓但仍保持50%+，毛利率小幅下降至72-74%
- ASIC份额持续提升，竞争格局多元化
- 数据中心Capex增速从70%+降至30-40%

**投资含义：** NVIDIA估值可能见顶；AMD/博通等二线受益者估值提升；HBM厂商利润率承压

### 情景C：需求放缓/供给过剩（概率20%）

**假设：** AI商业化回报低于预期，云服务收入增速降至15%以下，资本开支回报周期拉长

**推演：**
- 2027年可能出现CoWoS/HBM产能过剩
- NVIDIA增速大幅放缓至20-30%，毛利率下降至70%以下
- 数据中心Capex增速降至10-20%
- 部分AI初创公司因算力成本过高而退出

**投资含义：** 科技巨头估值回调；供应链厂商利润率压缩；周期性调整


- 立即受益：已有有效产能、通过认证且订单可转收入的节点。
- 滞后受益：等待扩产、良率爬坡、并网或客户认证的节点。
- 可能非受益：只有规划产能、概念关联或无法转嫁成本的节点。
